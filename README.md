## OIBSIP – Level 2 Task 4: Login Authentication System  

### About  
This project is part of my Oasis Infobyte Web Development Internship (Level 2 – Task 4).  
It is a simple Login and Registration System built using HTML, CSS, and JavaScript.  
Users can register with credentials, log in using stored data, and switch between login and signup forms dynamically.  

### Features  
- Registration and login forms with validation  
- Local storage for user credential management  
- Dynamic form toggle (Login ↔ Register)  
- Error and success message handling  
- Modern glassmorphism UI with background image and gradient overlay  

### Tech Stack  
- HTML5  
- CSS3  
- JavaScript
